
git setting
============

본인이 연습할 terraform 깃 저장소를 생성합니다.
커맨드는 다음과 같습니다.

### Git 생성 & gitignore setting
------------

> git clone <본인 리파지토리 경로>  // git clone https://github.com/azjaehyun/terra-middle-class.git
>
> touch .gitignore  //  해당파일에 .terraform *.tfstate *.tfstate.backup 3줄 추가
> 
> git commit -m " init commit"
>
